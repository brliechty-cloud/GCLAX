🔑 GitHub Token (one-time setup)

To save changes live to the site, you need a GitHub token:
1. Go to github.com/settings/tokens/new
2. Give it a name (e.g. "GC Lax Admin")
3. Set expiration to No expiration
4. Check the repo scope box
5. Click Generate token, paste it below
